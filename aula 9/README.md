## Aula 9 - Typescript e Superset

### TypeScript é JavaScript

O Typescript é um superset, que adiciona a sintaxe de tipos ao JavaScript, que por sua vez corrigi pontos de fragilidade que apenas a linguagem original possui, lembrando que o TypeScript é convertido para JavaScript, por isso onde roda um roda o outro.

#### Transpile

O TypeScript é traduzido e compilado para JavaScript

### Superset

Superset é a adição de conceitos e ferramentas a uma linguagem de programação, formando uma nova apartir dessas adições.

### Runtime x Compile Error

#### Runtime

Erros de Runtime, acontecem apenas com as aplicações rodando, a maioria dos erros JS acabam por se enquadrar nesse quadrante. O que se torna inviável para aplicações maiores


#### Compile

Capturar erros antes da compilação, o que a própria IDE te avisa sobre o erro avistado, assim tendo uma camada extra de segurança.
